# Module 1 : Généralités sur les systèmes informatiques

## Séquence 1 : Définitions et vocabulaire de base

### 1- Information

#### 1.1- Définition

<div class="rounded-2xl border-2 border-blue-500 bg-gray-100 p-2">

L'**information** est un ensemble de données ayants un sens comrpéhensible par
l'homme

Donc :

{" "}

<div class="rounded-xl border-2 border-blue-500 bg-white p-2 text-center text-xl font-bold">
  Information = Données + Sens
</div>

</div>
#### 1.2- Types d'informations

Les informations peuvent prendre plusieurs formes, notamment :

```mermaid
graph TD
    A(Information) --> B(Données textuelles)
    A(Information) --> C(Images)
    A(Information) --> D(Vidéos)
    A(Information) --> E(Sons)
    A(Information) --> F(...etc)
```

#### 1.3- Support et Source d'information

Les informations sont stockées et transmises à travers divers supports et sources. Voici quelques exemples :

- Un **Supports** d’information est le moyen qui permet de la conserver.
  Ordinateur, Journal, caméra, CD-ROM, portable, ….
- La **source** de l’information est le moyen qui permet de la transmettre.
  Télévision, radio, internet…

```mermaid
graph LR
    A[Information] --> B[Supports d'information]
    A[Information] --> C[Sources d'information]
    B --> D(Journal,caméra,CD-ROM, Portable)
    C --> F(TV, Radio,Internet ...)
```

### 2- Traitement

#### 2.1- Définition

Le **traitement** peut être défini comme un ensemble d'opérations qui permet de transformer des données d'entrée en un résultat en suivant des règles définies.

```mermaid
graph LR
    subgraph Traitement
        style TraitementNode fill:#F9EBAE,stroke:#F2AB27
        ÉtatInitial --> TraitementNode[Traitement<br\> Ensemble d'opérations]
        TraitementNode --> ÉtatFinal
        style ÉtatInitial fill:#8ED8B3,stroke:#32A34C
        style ÉtatFinal fill:#8ED8B3,stroke:#32A34C
        ÉtatInitial
        ÉtatFinal
    end

```

**Exemple :**

```mermaid
graph LR
  A(<b>État Initial:</b><br/>Orange + Sucre) --> B(Traitement )
  B --> C(<b>État Final:</b><br/>Jus d'orange)
```

#### 2.2- Types du Traitement

```mermaid
graph TD
  style F fill:#F7EBEC,stroke :#DCA7AC
```

### 3- Définition de l'informatique

<div class="rounded border-2 border-blue-500 bg-blue-50 p-3 text-center text-2xl font-bold ">
  Informatique = **Infor**mation + Auto**matique**
</div>
**L’informatique** :{" "}

> Est la science du traitement automatique de l’information

Automatique signifier sans intervention humaine.
